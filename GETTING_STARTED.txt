QUICK START (5 MINUTES)
================================================================================

1. INSTALL DEPENDENCIES
   npm install

2. SET UP POSTGRESQL
   # Make sure PostgreSQL is installed and running
   # Create a database named 'jobportal'
   
3. CONFIGURE ENVIRONMENT
   # The .env file is already set up with defaults
   # Update DATABASE_URL if your PostgreSQL has different credentials
   
4. RUN DATABASE MIGRATION
   npm run db:migrate
   # When prompted, name it "init"
   
5. START DEVELOPMENT SERVER
   npm run dev
   
6. OPEN YOUR BROWSER
   Visit: http://localhost:3000

================================================================================
CREATING YOUR FIRST ACCOUNTS
================================================================================

1. REGISTER AS A JOB SEEKER
   - Go to http://localhost:3000/register
   - Select "Job Seeker"
   - Fill in details and register

2. REGISTER AS A COMPANY
   - Sign out
   - Register again, this time select "Company"
   - Fill in company details

3. CREATE AN ADMIN ACCOUNT
   - Register normally (as job seeker or company)
   - Then run this SQL command:
   
   psql postgresql://postgres:postgres@localhost:5432/jobportal
   
   UPDATE users SET role = 'ADMIN' WHERE email = 'your@email.com';
   
   \q

================================================================================
ENABLING PAYMENT (OPTIONAL)
================================================================================

To enable actual payments:

1. Create a Stripe account at stripe.com
2. Get test API keys from Dashboard
3. Update .env with your keys:
   STRIPE_PUBLIC_KEY="pk_test_..."
   STRIPE_SECRET_KEY="sk_test_..."

4. Set up local webhook testing:
   stripe listen --forward-to localhost:3000/api/payments/webhook
   
5. Copy webhook secret to .env:
   STRIPE_WEBHOOK_SECRET="whsec_..."

6. Test with card: 4242 4242 4242 4242

================================================================================
ENABLING EMAIL (OPTIONAL)
================================================================================

To enable email notifications:

1. Create account at resend.com
2. Get API key
3. Update .env:
   RESEND_API_KEY="re_..."
   EMAIL_FROM="onboarding@resend.dev"
